Web服务请求处理步骤

一次完整的http请求处理过程

Web访问响应模型（Web I/O）

  • 单进程I/O模型：启动一个进程处理用户请求，而且一次只处理一个，多个请求被串行响应
  • 多进程I/O模型：并行启动多个进程,每个进程响应一个连接请求
  • 复用I/O结构：启动一个进程，同时响应N个连接请求实现方法：多线程模型和事件驱动多线程模型：一个进程生成N个线程，每线程响应一个连接请求事件驱动：一个进程处理N个请求
  • 复用的多进程I/O模型：启动M个进程，每个进程响应N个连接请求，同时接收M*N个请求
